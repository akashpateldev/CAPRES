# CAPRES
CAPRES (Centralized Academic Project Repository &amp; Evaluation System) is a department-level web platform developed for my college to manage B.Tech final-year projects. It supports secure submission, faculty evaluation, group projects, and archived learning using registration-number-based identity and role-based authentication.

A centralized platform for academic project submission, evaluation, and innovation tracking. 🏆 Winner – Startup Bootcamp 2026 (FTBI, NIT Rourkela).
